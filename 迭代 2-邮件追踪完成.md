### 3. 邮件效果统计

#### 新增统计指标
| 指标 | 说明 | 计算公式 |
|------|------|----------|
| 已发送 | 总发送数量 | - |
| 已打开 | 至少打开 1 次的邮件数 | - |
| 已点击 | 至少点击 1 次的邮件数 | - |
| 已回复 | 客户回复的邮件数 | - |
| 打开率 | 打开的百分比 | 打开数/发送数 |
| 点击率 | 点击的百分比 | 点击数/发送数 |
| 回复率 | 回复的百分比 | 回复数/发送数 |

#### 当前演示数据
```
总发送：3 封
已打开：3 封 (100% 打开率)
已点击：1 封 (33% 点击率)

```

## 💡 技术亮点

### 1. 追踪像素原理
```html
<!-- 邮件中嵌入 -->
<img src="https://track.example.com/api/track/open/trk_abc123" 
     width="1" height="1" style="display:none"/>
```

当客户打开邮件时：
1. 邮件客户端加载图片
2. 服务器记录请求（User-Agent、IP、时间）
3. 返回 1x1 透明 PNG

---

### 2. 链接重定向原理
```html
<!-- 原始链接 -->
<a href="https://example.com/product">产品</a>

<!-- 追踪链接 -->
<a href="https://track.example.com/api/track/click/trk_abc123?url=https://example.com/product">产品</a>
```

当客户点击时：
1. 先访问追踪服务器
2. 记录点击事件
3. 302 重定向到目标 URL

---

### 3. User-Agent 解析
```javascript
parseUserAgent(userAgent) {
  const isMobile = /mobile|android|iphone|ipad/i.test(userAgent);
  const isWindows = /windows/i.test(userAgent);
  const isChrome = /chrome/i.test(userAgent);
  
  return {
    type: isMobile ? 'mobile' : 'desktop',
    os: isWindows ? 'Windows' : 'macOS',
    browser: isChrome ? 'Chrome' : 'Safari'
  };
}
```
