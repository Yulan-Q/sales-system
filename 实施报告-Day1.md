### 3. 智能线索评分（1.5 小时）

#### 评分维度（100 分制）
| 维度 | 权重 | 说明 |
|------|------|------|
| 📧 邮箱域名 | 20 分 | 公司域名 vs 免费邮箱 |
| 🏢 公司规模 | 30 分 | 员工数分级打分 |
| 🏭 行业匹配 | 25 分 | 目标行业匹配度 |
| 🌍 地理位置 | 15 分 | 目标国家/地区 |
| 💻 在线活跃 | 10 分 | 是否有网站 |

#### 评分等级
- **A 级 (80-100 分)**: 高质量线索，优先跟进

- **C 级 (40-59 分)**: 低质量，选择性跟进
- **D 级 (0-39 分)**: 极低质量，暂不跟进

### 4. 公海池机制（1 小时）

#### 核心规则
- ✅ 未分配线索自动进入公海
- ✅ 销售可领取公海线索
- ✅ 48 小时未跟进自动回收
- ✅ 管理员可手动回收

#### 数据库设计
```sql
ALTER TABLE leads ADD COLUMN (
  is_in_public_pool BOOLEAN DEFAULT FALSE,
  assigned_to INTEGER,
  auto_recycle_at TIMESTAMP
);
```

#### API 端点
```javascript
// 领取公海线索
POST /api/public-pool/claim
Body: { "leadId": 4 }

// 回收线索（定时任务触发）
POST /api/leads/:id/recycle
```

#### 业务价值
- 防止销售"占着茅坑不拉屎"
- 提高线索利用率
- 促进团队内部竞争
