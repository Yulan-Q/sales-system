## 🎯 访问演示

### Docker 部署（推荐）

```bash
# 一键启动
git clone https://github.com/Yulan-Q/sales-system.git
cd sales-system
cp .env.example .env
docker-compose up -d
```

**访问地址**:
- 前端：http://localhost:8080
- 后端：http://localhost:3000
- 数据库：localhost:5432

### 本地开发

```bash
# 后端
cd backend
node core-server-v2.4.js

# 前端
cd frontend
python3 -m http.server 8080
```

**测试账号**: `admin` / `admin123`

### 对销售团队
1. **效率提升 10 倍** - AI 写信 30 秒 vs 手动 30 分钟
2. **转化率提升 3 倍** - 智能评分优先跟进高价值
3. **工作量减少 80%** - 批量操作自动化

### 对管理层
1. **数据安全** - JWT 认证 + 权限控制
2. **决策依据** - 实时数据仪表盘
3. **资源优化** - 公海池防止浪费

### ROI 计算
```
假设:
- 销售月薪：¥10,000
- 每天写信：10 封 × 30 分钟 = 5 小时
- AI 写信：10 封 × 30 秒 = 5 分钟
- 节省时间：5 小时 - 5 分钟 = 4.9 小时/天

月度节省:
- 时间：4.9 小时 × 22 天 = 108 小时
- 成本：108 小时 ÷ 176 小时 × ¥10,000 = ¥6,136

年节省：¥6,136 × 12 = ¥73,632/人

系统价格：¥9,999/年
ROI: (¥73,632 - ¥9,999) / ¥9,999 = 636%
```
