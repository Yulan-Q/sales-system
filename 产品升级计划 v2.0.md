### 二、AI 核心增强 (The "AI" in the System)

#### 2.1 智能线索评分 (Lead Scoring) 🎯
**功能描述**: AI 分析线索质量，自动打分 0-100

**评分维度**:
- [ ] 邮箱域名质量（公司域名 vs 免费邮箱）+20 分
- [ ] 公司规模（员工数/营收）+30 分
- [ ] 行业匹配度 +25 分
- [ ] 地理位置（目标市场）+15 分
- [ ] 在线活跃度（网站/社媒）+10 分

**UI 展示**:
- [ ] 线索列表显示评分徽章
- [ ] 按评分排序/筛选
- [ ] 高分线索自动置顶
- [ ] 评分详情弹窗（解释得分原因）

**技术实现**:
```javascript
// 评分算法伪代码
function calculateLeadScore(lead) {
  let score = 0;
  
  // 邮箱域名
  if (isCompanyDomain(lead.email)) score += 20;
  
  // 公司规模
  if (lead.employees > 50) score += 30;
  else if (lead.employees > 10) score += 15;
  
  // 行业匹配
  if (matchTargetIndustry(lead.industry)) score += 25;
  
  // 地理位置
  if (TARGET_COUNTRIES.includes(lead.country)) score += 15;
  
  // 在线活跃
  if (hasActiveWebsite(lead.website)) score += 10;
  
  return Math.min(score, 100);
}
```

**优先级**: ⭐⭐⭐⭐⭐ (核心差异化功能)  
**预估工时**: 16 小时

#### 3.3 时区智能发送 🌍
**功能描述**: 根据线索时区自动选择最佳发送时间

**逻辑实现**:
```javascript
function calculateSendTime(lead) {
  const leadTimezone = getTimezoneByCountry(lead.country);
  const bestHour = 9.5; // 上午 9:30
  
  const leadLocalTime = moment().tz(leadTimezone);
  const sendTime = leadLocalTime.hour(bestHour).minute(0);
  
  return sendTime.toDate();
}
```

**最佳发送时间规则**:
| 国家 | 最佳时间 | 备注 |
|------|----------|------|
| 美国 | 9:30 AM EST | 避开周一早晨 |
| 英国 | 9:30 AM GMT | 周二 - 周四最佳 |
| 德国 | 10:00 AM CET | 避开午餐时间 |
| 日本 | 10:00 AM JST | 周一上午适合 |

**功能扩展**:
