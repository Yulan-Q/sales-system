## 📋 下一步行动

### 1. 注册 API Keys（必须）
- **Apify**: https://console.apify.com/account#/integrations
- **Brevo**: https://app.brevo.com/settings/keys/api

### 2. 更新配置
编辑 `backend/.env` 文件：
```env
APIFY_TOKEN=你的真实 Apify Token
BREVO_API_KEY=你的真实 Brevo API Key
```

### 3. 启动完整版后端
```bash
cd backend
# 停止演示版
pkill -f demo-server.js

# 启动完整版（需要修复所有依赖）
node src/index.js
```

### 4. 部署前端 Vue 应用
```bash
cd frontend
npm run build
# 或使用 Vite 开发服务器
npm run dev
```
