## 🚀 快速开始

### 1. 启动后端

```bash
cd backend
node core-server-v2.3.js
```

### 2. 启动前端

```bash
cd frontend
python3 -m http.server 8080
```

### 3. 访问系统

浏览器打开：http://localhost:8080/v2.3-dashboard.html
