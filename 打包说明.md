## 🚀 快速启动（5 分钟）

### 步骤 1: 安装依赖
```bash
cd /home/admin/openclaw/workspace/delivery/企业级 AI 外销系统

# 后端
cd backend
npm install

# 前端
cd ../frontend
npm install
```

### 步骤 2: 配置环境变量
```bash
# 复制环境变量模板
cp .env.example .env

# 编辑 .env 文件，填入：
# - APIFY_API_TOKEN=你的 Apify Token
# - BREVO_API_KEY=你的 Brevo API Key
# - JWT_SECRET=生成的随机密钥
```

### 步骤 3: 启动服务
```bash
# 方式 1: 开发模式（前后端分开启动）
cd backend && npm run dev
cd frontend && npm run dev

# 方式 2: Docker 一键启动（推荐）
docker-compose up -d
```

### 步骤 4: 访问系统
- 前端：http://localhost:5173
- 后端 API: http://localhost:3000
- 测试账号：demo / demo123
